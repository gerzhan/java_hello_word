# java_hello_word

## Work with application

### Compile application

```bash
$javac HelloWordApplication.java
```

### Run application

```bash
$java HelloWordApplication
```

## Work with documentation

### Instal dependency

- install [docsify](https://docsify.js.org/)

```bash
$npm i -g docsify-cli
```

### Run documentation

```bash
$docsify serve docs
```
